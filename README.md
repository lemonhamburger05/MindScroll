# 📌 MindScroll

**MindScroll** 是一款专为 macOS 设计的极简待办事项管理工具。它巧妙地将任务列表和 DDL（截止时间）以“歌词滚动”的形式驻留在 Mac 的顶部菜单栏中。
无论你打开了多少个窗口，当前最重要的任务始终在你抬眼可见的地方，有效减少应用切换带来的注意力分散，助你保持心流状态。

## ✨ 核心功能 (Features)

* 🎵 **歌词式滚动提示**：任务与 DDL 在 Mac 顶部菜单栏匀速滚动，提供温和且持续的强提醒。
* 🖱️ **原生级无缝交互**：深度集成 macOS API，点击菜单栏滚动的文字即可一键唤出或隐藏控制面板。
* 📝 **精细化任务管理**：支持自定义任务名称与 DDL；提供一键“完成”和弹窗式的“延期”修改功能。
* ↕️ **直观的拖拽排序**：控制面板内的任务列表支持鼠标按住拖拽（Drag and Drop），轻松调整任务优先级。
* ⚙️ **高度可定制化**：支持用户自定义菜单栏的**显示文字长度**与**滚动速度**，满足不同屏幕尺寸和阅读习惯。
* 💾 **本地数据持久化**：自动将任务列表和用户偏好设置保存至本地 JSON 文件，关闭重启后无缝恢复，随开随用。

## 🛠️ 环境配置 (Prerequisites)

本项目基于 Python 开发，使用了 `PyQt6` 构建用户界面，并通过 `PyObjC` 调用 macOS 底层原生菜单栏接口。

**运行环境要求：**
* macOS 操作系统
* Python 3.8 或更高版本

**安装依赖项：**
请在终端中运行以下命令以安装所需的第三方库：
```bash
pip install PyQt6 pyobjc-core pyobjc-framework-Cocoa

## 🚀 安装与运行 (Installation & Usage)
  1. 克隆仓库到本地：
  git clone [https://github.com/你的用户名/MindScroll.git](https://github.com/你的用户名/MindScroll.git)
  cd MindScroll

  2. 运行程序：
  python main.py

  3. 使用指南：
  程序启动后，你的 Mac 顶部菜单栏会出现 📌 点击添加待办 的字样。

  点击该字样，屏幕中央会弹出 MindScroll 的设置面板。

  在输入框中填入待办事项和对应的 DDL，点击“添加”。

  如有需要，可在下方调整“菜单栏显示长度”和“滚动速度”。

  按住列表中的任务可以上下拖动排序。

  点击 “保存配置并开始滚动”，面板将自动隐藏，你的任务开始在顶部菜单栏滚动！
